# Roadmap de Melhorias - Site SDJ

Este documento descreve as futuras melhorias planejadas para o site da SDJ – Serviços de Desenvolvimento de Jogos.

## Curto Prazo
- [x] **Otimização de SEO**: Adicionar meta tags description, keywords e Open Graph para melhor compartilhamento em redes sociais.
- [x] **Favicon**: Criar e adicionar um ícone personalizado para a aba do navegador.
- [ ] **Google Analytics**: Integrar para monitoramento de tráfego.

## Médio Prazo
- [x] **Seção de Portfólio**: Criar uma página dedicada com galeria de jogos desenvolvidos (imagens/vídeos).
- [ ] **Blog**: Implementar um blog para compartilhar novidades da empresa e artigos sobre desenvolvimento de jogos.
- [x] **Modo Escuro**: Adicionar toggle para alternar entre tema claro e escuro (respeitando preferência do sistema).

## Longo Prazo
- [ ] **Backend de Contato**: Substituir a simulação de envio por um backend real (Node.js ou PHP) ou serviço como Formspree.
- [ ] **Área do Cliente**: Portal para clientes acompanharem o progresso de seus projetos.
- [ ] **Internacionalização (i18n)**: Adicionar suporte para inglês e espanhol.
